# Bugfix Issue (PDD)

## 1. Problem
Was ist der beobachtete Fehler?  
Was sollte passieren, was passiert stattdessen?

## 2. Prüfbarkeit
- Wie lässt sich der Fehler reproduzieren?
- Welche Kriterien definieren „behoben“?

## 3. Annahmen
- Welche Systemannahmen gelten?
- Welche Eingaben / Zustände sind relevant?

## 4. Verantwortung
- Welche Diagnose triffst du selbst?
- Was darf die KI analysieren oder vorschlagen?

## 5. Kontext
Logs, Screenshots, Beispiele, relevante Codebereiche.

## 6. Risiko
Wie kritisch ist der Bug? (low / medium / high)

## 7. Done Criteria
- [ ] Reproduzierbar verstanden  
- [ ] Ursache identifiziert  
- [ ] Fix spezifiziert  
- [ ] Fix validiert  
- [ ] Regression ausgeschlossen
